Web Server Plugin

This is a built-in extension plugin that follows the plugin template system.
The web_server plugin is a plugin that introduces a web interface for the bot along with REST APIs for custom implementations.
The web_server settings can be modified in the metadata.ini file or by using launch parameters.

Usage:

  • !startwebserver: Starts the web server instance if it is not already running.
  • !stopwebserver: Stops the web server instance if it is running.
