Web Server Plugin

Usage:

  • !startwebserver: Starts the web server instance if it is not already running.
  • !stopwebserver: Stops the web server instance if it is running.
Plugin Metadata

    PluginVersion = 5.1.0

    PluginDescription = The Web Server plugin handles REST+WebSocket+WebApp functionality.
    PluginLanguage = EN
    PluginCommands: [

                    "startwebserver"
