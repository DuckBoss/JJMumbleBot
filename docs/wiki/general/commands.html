Bot Commands System

Simple Commands:

Users can enter commands to the bot by typing the Command Token before a command like so:
!help --> This command shows the list of available plugin help content.
!help 'plugin_name' --> This command shows help commands for individual plugins.
!version --> This command shows the current bot version.
Please note that commands are CASE-SENSITIVE

Multi-Commands:

There is also support for multi-command inputs which allows you to input multiple commands in a single line like so:
!version ; !about ; !echo test --> This multi-command calls the !version, !about, and !echo commands.
!ytsearch songname ; !p 1 --> This multi-command searches youtube for a song and play the first result.
You can set the multi-command limit (commands per input) in the config.ini

Custom Command Tokens:

You can also change the command token to any single character in your config.ini file.
For example, you can change the token to '~' or '/' so your commands will look like this:
~help --> This command shows help commands.
/version --> This command shows the current bot version.

Primary Bot Control Plugins: Core Commands and Bot Commands

Core Commands:

A List Of Core Commands:
!sleep 'seconds' --> The command queue sleeps for the given amount of seconds.
!version --> Retrieves the current version of the bot.
!about --> Retrieves the bot's description provided in the config.ini file.
!uptime --> Displays the amount of time the bot has been online.
!exit --> Quits all plugins and shuts down the bot.
!restart --> Completely stops the bot and restarts it.
!restartplugins --> Quits and restarts all active plugins.
!setcomment 'comment_text' --> Sets the bot's user comment with the given text.
!pguistresstest --> Conducts a stress test of the PGUI system in the channel chat. Uses a default of 5 lines.
!pguistresstest 'num_of_lines' --> Conducts a stress test of the PGUI system in the channel chat. Optionally set the number of lines to control the size of the test.
!resetcomment --> Resets the bot's user comment to the default text.
!help --> Displays all the plugins available with help content.
!help 'plugin_name --> Displays all the commands available for the given plugin.
!setalias 'alias_name' 'command'|'command'|... --> Registers an alias for single/multi-commands.
!aliases --> Echoes a list of all registered aliases in the channel chat.
!removealias 'alias_name' --> Removes the registered alias.
!clearaliases --> Removes all registered aliases.
!history --> Displays the command history.
!clearhistory --> Clears the command history.

Bot Commands:

A List Of Bot Commands:
!echo 'message/image' --> Echoes a message/image in the chat.
!msg 'username' 'message' --> Anonymously private messages a user from the bot.
!log 'message' --> Manually logs a message by an administrator.
!showplugins --> Displays a list of all the active plugins.
!makechannel 'channel_name' --> Creates a channel with the given name.
!move 'channel_name' --> Moves to an existing channel with the given name.
!move default/Default --> Moves the bot to the default channel stated in the config.ini file.
!joinme --> Moves to the users channel.
!joinuser 'username' --> Moves the bot to the specified users channel.
!leave --> Moves the bot to the default channel.
!showprivileges --> Displays the full user list with privilege levels.
!setprivileges 'username' 'level' --> Sets a user's privilege level.
!addprivileges 'username' 'level' --> Adds a new user to the user privilege list.
!blacklistuser --> Displays the current list of users in the blacklist.
!blacklist 'username' --> Blacklists specific users from using certain plugin commands.
!whitelistuser 'username' --> Removes an existing user from the blacklist.
