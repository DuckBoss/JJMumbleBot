All commands can be run by typing it in the channel or privately messaging JJRadioBot.
!rnew 'stream link': Use this command to tune into a radio stream.
!rdefault: Streams the first radio link from the available radio links.
!rvolume/!v '0..1': Sets the radio plugin audio volume.
!rstop: Stops the current radio stream.
!rnext: Goes to the next radio stream from the available radio links.
