All commands can be run by typing it in the chat or privately messaging JJMumbleBot.
!setstatus 'message': Sets the users status message.
!status 'username': Displays the given users status message.
!mystatus: Displays the users status message.
!clearstatus: Clears the users status message.
!announcement: Sets a global announcement.
!setannouncement: Gets the current global announcement.
!clearannouncement: Clears the current global announcement.
