All commands can be run by typing it in the channel or privately messaging JJMumbleBot.
!sb 'file_name': Plays a locally saved '.wav' audio file.
!sbnow/!sbn 'file_name': Plays a locally save '.wav' audio file and overrides queueing.
!sbquiet/!sbq 'file_name': Plays a locally saved '.wav' file without displaying what is being played.
!sbquietnow/!sbqn 'file_name': Plays a locally saved '.wav' file without displaying what is being played and overrides queueing.
!sbrandom/!sbr: Plays a random sound clip from the list of available clips.
!sbrandomnow/!sbrn: Plays a random sound clip from the list of available clips and overrides queueing. !sblist/!sbl: Displays all the available sound board tracks in private messages.
!sbdownload 'youtube_url' 'file_name': Downloads a sound clip from a youtube link and adds it to the sound board.
!sbdelete 'file_name.wav': Deletes a clip from the sound board storage. Be sure to specify the .wav extension.
