All commands can be run by typing it in the channel or privately messaging JJMumbleBot.
!ytsearch 'search_term': Searches youtube for a song/video.
!ytplay 'item_number' 'item_count'(optional): Plays the selected song from youtube.
!ytlink 'youtube_link': Adds the given youtube link to the queue.
!ytlinkfront 'youtube_link': Priority inserts the given link to the front of the queue.
!ytplaylist 'youtube_playlist': Adds the given youtube playlist to the queue.
